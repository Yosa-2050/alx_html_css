# Headphones Webpage

This README describes the Headphones webpage project for ALX.

## Project Overview

The goal of this project is to create a responsive webpage that matches a Figma design of a headphone landing page.

## Design Details

- Responsive design: switches to mobile layout at screen width 480px or less.
- Fonts used: Source Sans Pro and Spin Cycle OT.
- Links hover/active color: #FF6565.
- Button hover/active opacity: 0.9.
- Maximum content width: 1000px, centered.

## Notes

- Some values in the Figma design are floats; they can be rounded during implementation.
- Custom fonts need to be installed if not present on the system.
